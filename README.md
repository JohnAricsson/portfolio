# 🚀 Portfolio

My personal modern developer portfolio built with **Next.js**, **Tailwind CSS**, **Aceternity UI** and **Sanity CMS**.

🌐 **Live:** https://johnaricsson.vercel.app

## ✨ Features

* Fully responsive design
* Dynamic content powered by Sanity CMS
* Interactive 3D project showcases
* Animated academic timeline
* Apple-inspired experience carousel
* Dark & light mode support
* Optimized for performance and SEO

## 🛠️ Tech Stack

* Next.js
* React
* Tailwind CSS
* TypeScript
* Sanity CMS
* Vercel

## 🎯 Purpose

A personal space to showcase my projects, achievements, experience, and journey as a developer—all in a fast, modern, and interactive experience.
